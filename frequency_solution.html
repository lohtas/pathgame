Frequency Distribution Graph Solution


While using a computer icon as a data point ties the graph to the theme of the data, it does not allow for an accurate reading of the graph. Using a circled dot is a bit more effective in providing accuracy, but still uses excessive graphics that serve no purpose. Using simple shapes, such as a dot, square, triangle, is more accurate and efficient. For more information on Tufte's ideas on minimizing data ink, see Boyd's (2000) Chapter 2: "The Gospel According to Tuft".
