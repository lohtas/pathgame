With the elements provided, it is possible to configure the graph in more than one way. Both lines (with or without points) or bars are appropriate for this graph, depending on the type of information you want to convey. Using bars is a good method to compare how each item measures up to each other each week. However, using lines in this way would serve two functions: viewers can focus on the circulation pattern of each item type over four weeks (Do the number of checkouts increase or decline?) as well as see how each item compares to the other each week, as long as each item's line is distinguished in some way. In this case, different colors represent each item and a legend is provided.


While it is acceptable to use either the set of lines or set of bars, it is important to place the categorical and quantitative scales on the appropriate axes. According to Few (2005), it is more intuitive to place interval scales (or time-series) horizontally, rather than vertically. The bars and lines were drafted with the intent of placing the number of checkouts on the y-axis (left side) and weeks on the x-axis (bottom). Because each color of the lines and bars correspond to an item type, the item type labels are not necessary.
